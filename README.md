# gif-splitter
gif画像をフレームに分解し、canvas上で再生します。<br>
JavaScriptを使ったゲームやアニメーションの素材にgif画像を素材として使いたい時用です。<br>
バックエンドはJavaサーブレットを使っています。<br>
フロントエンドはadobe社のCREATEJSを使用しています。<br>
<br>
